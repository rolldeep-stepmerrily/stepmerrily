## Description
음악 테마의 커뮤니티 입니다.
  
게시글, 댓글 작성 및 추천 기능이 제공되고 프로필 사진, 뮤직, 상태메시지 기능으로 타인과 소통할 수 있습니다.
  
플레이리스트를 생성하여 타인과 공유하고, 자신이 보유한 악기를 등록해보세요!
